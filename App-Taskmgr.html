1000-Introduction


INTRODUCTION TO SymbOS Task Manager

With the task manager you can have a view about the utilization of the system,memory uses and process and applications. There is also the possibility to start are stop applications or processes..

1100-Feature overview


FEATURES

You can start or end applications using the task manager.


2000-Using the task manger


Using the task manger
Choose a application or a process that you want to stop or kill. The best way to end a application is stop the application by using the application itself. How ever you can force SymbOS to kill an application or process if it is not responding anymore. You can also view the performance system. Change the view if you want to see it in a higher or lower speed.

Enjoy the multitasking in SymbOS on a Z80!
